AuraFX - Professional MythicMobs Particle Effect Generator

Create Advanced Particle Effects for Minecraft Servers

AuraFX is a comprehensive web-based tool designed specifically for Minecraft server owners and plugin developers who use MythicMobs. Our professional particle effect editor enables you to create stunning visual effects that enhance your server's gameplay experience.

Key Features and Capabilities

Visual Editor Interface

Design particle effects using an intuitive drag-and-drop interface. Our canvas-based editor allows you to create complex patterns, shapes, and animations with ease. The visual approach eliminates the need for manual YAML coding while providing precise control over every aspect of your effects.

3D Editor Integration

Take your effects to the next dimension with our integrated 3D editor. Create three-dimensional particle systems, rotate and manipulate effects in real-time, and preview how your creations will look from any angle. The 3D workspace operates independently while maintaining synchronization with the main 2D canvas.

Layer-Based Effect System

Build complex effects by combining multiple layers. Each layer can have different particle types, colors, sizes, and behaviors. Our advanced layer system supports various effect types including rings, spheres, tornados, orbitals, and custom shapes. Reorder layers with drag-and-drop functionality and toggle visibility for testing.

Animation Modes and Presets

Bring your effects to life with our comprehensive animation system. Choose from global animation modes that affect all particles, or apply layer-specific animations. Features include rotation, movement, rainbow effects, chain reactions, mirror effects, and performance optimization modes.

Import and Export Capabilities

Import particle data from PNG images or 3D OBJ models. Our PNG importer converts pixel data into particles with customizable size, threshold, and tolerance settings. The OBJ importer extracts vertices from 3D models, perfect for creating effects based on custom shapes and structures.

Automatic Code Generation

Generate MythicMobs-compatible YAML code automatically. Our system creates optimized particle effect configurations that work seamlessly with the MythicMobs plugin. No manual coding required - simply design your effect visually and export the ready-to-use code.

Professional Tools and Utilities

Access a complete suite of professional tools including color pickers, grid snapping, shape management, and real-time preview. Our performance mode ensures smooth operation even with complex effects, while the integrated code panel provides instant feedback on your YAML output.

Code Generation (MythicMobs)

Export MythicMobs YAML with particles, colors, and offsets from your scene.

Effect Groups

The effect grouping system in AuraFX's code generation process intelligently organizes vertices by particle type or effect characteristics, ensuring that the resulting MythicMobs YAML output is well-structured, readable, and optimized for performance. At export time, the system automatically analyzes the scene and groups vertices based on their particle type, visual properties, and behavioral characteristics, creating logical divisions that make the generated code both organized and compact. This intelligent grouping significantly reduces parameter duplication by consolidating similar elements under shared effect definitions, resulting in cleaner, more maintainable YAML files. The grouping process creates natural breakpoints within the exported file, making it easier to read, understand, and modify the generated code during manual review or post-processing. Each group includes comprehensive comments that provide counts, timestamps, and descriptive information for enhanced traceability and debugging capabilities. The system's ability to preserve element identifiers across multiple exports ensures that diffs between different versions remain stable and meaningful, which is particularly valuable during code review processes when tracking changes to timing, density, or other effect parameters. This stable diffing capability enables effective version control and collaborative development workflows, as team members can easily identify and understand changes between different iterations of the same effect. The grouping system also includes advanced features such as custom grouping rules, priority-based organization, and intelligent parameter optimization that further enhance the quality and efficiency of the generated code.

Offsets & Precision

The coordinate system and precision management in AuraFX's code generation ensures that exported MythicMobs YAML files are optimized for both performance and maintainability. Coordinates are emitted as x, y, and z offsets relative to the origin point, with precision automatically clamped to a sensible number of decimal places that balances accuracy with file size efficiency. This intelligent precision management avoids unnecessary noise in version control systems and keeps the generated file size under control while preserving the overall visual shape and integrity of the effect. The system employs sophisticated algorithms to determine the optimal precision level for each coordinate based on the scale and complexity of the effect, ensuring that important details are preserved while eliminating redundant precision that could impact performance. Colors are written in a standardized format that ensures compatibility across different MythicMobs versions and configurations, while opacities and transparency values are accurately preserved to match the layer settings from the original design. The flat schema structure of the generated YAML ensures that each instruction is independent and can be safely modified without affecting other elements, making post-export editing straightforward and risk-free. This independence is particularly valuable when fine-tuning effects for specific server configurations or when making targeted adjustments to individual elements within complex effects. The offset system also includes validation checks to ensure that generated coordinates fall within reasonable bounds and that the resulting effect will display correctly in the target environment. Advanced features include automatic coordinate optimization, intelligent rounding strategies, and configurable precision settings that allow creators to balance between accuracy and performance based on their specific requirements.
