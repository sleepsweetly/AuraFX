AuraFX - Professional MythicMobs Particle Effect Generator

Create Advanced Particle Effects for Minecraft Servers

AuraFX is a comprehensive web-based tool designed specifically for Minecraft server owners and plugin developers who use MythicMobs. Our professional particle effect editor enables you to create stunning visual effects that enhance your server's gameplay experience.

Key Features and Capabilities

Visual Editor Interface

Design particle effects using an intuitive drag-and-drop interface. Our canvas-based editor allows you to create complex patterns, shapes, and animations with ease. The visual approach eliminates the need for manual YAML coding while providing precise control over every aspect of your effects.

3D Editor Integration

Take your effects to the next dimension with our integrated 3D editor. Create three-dimensional particle systems, rotate and manipulate effects in real-time, and preview how your creations will look from any angle. The 3D workspace operates independently while maintaining synchronization with the main 2D canvas.

Layer-Based Effect System

Build complex effects by combining multiple layers. Each layer can have different particle types, colors, sizes, and behaviors. Our advanced layer system supports various effect types including rings, spheres, tornados, orbitals, and custom shapes. Reorder layers with drag-and-drop functionality and toggle visibility for testing.

Animation Modes and Presets

Bring your effects to life with our comprehensive animation system. Choose from global animation modes that affect all particles, or apply layer-specific animations. Features include rotation, movement, rainbow effects, chain reactions, mirror effects, and performance optimization modes.

Import and Export Capabilities

Import particle data from PNG images or 3D OBJ models. Our PNG importer converts pixel data into particles with customizable size, threshold, and tolerance settings. The OBJ importer extracts vertices from 3D models, perfect for creating effects based on custom shapes and structures.

Automatic Code Generation

Generate MythicMobs-compatible YAML code automatically. Our system creates optimized particle effect configurations that work seamlessly with the MythicMobs plugin. No manual coding required - simply design your effect visually and export the ready-to-use code.

Professional Tools and Utilities

Access a complete suite of professional tools including color pickers, grid snapping, shape management, and real-time preview. Our performance mode ensures smooth operation even with complex effects, while the integrated code panel provides instant feedback on your YAML output.

Perfect for Minecraft Server Enhancement

AuraFX is specifically designed for Minecraft server environments using the MythicMobs plugin. Create boss fight effects, spell animations, environmental effects, and custom abilities that will impress your players and enhance their gaming experience. Our tool streamlines the process of creating professional-quality particle effects without requiring extensive technical knowledge.

Technical Specifications

Built with Next.js 14 and modern web technologies, AuraFX provides a responsive and fast user experience. The application supports multiple languages, includes comprehensive documentation, and offers a wiki with detailed guides for all features. Our platform is optimized for both desktop and mobile devices.

Community and Support

Join our community of Minecraft server administrators and plugin developers. AuraFX is continuously updated with new features and improvements based on user feedback. Our comprehensive wiki provides detailed tutorials, examples, and best practices for creating amazing particle effects.

AuraFX is a professional-grade particle effect editor specifically designed for MythicMobs integration, offering an intuitive visual interface for creating complex particle animations. The editor combines real-time 2D canvas editing with advanced 3D preview capabilities, allowing creators to design effects through direct manipulation while maintaining precise control over particle behavior, timing, and spatial relationships. Built with performance in mind, AuraFX handles thousands of particles efficiently through optimized rendering pipelines and intelligent level-of-detail systems, making it suitable for both simple effects and complex multi-layered animations that would be challenging to create through traditional code-based approaches.

Visual Editor

The visual editor provides an intuitive drag-and-drop interface that allows creators to place particles, shapes, and effects directly on a responsive canvas with real-time feedback. This approach eliminates the need for complex coordinate calculations and YAML syntax memorization, making particle effect creation accessible to both technical and non-technical users. The editor features intelligent snapping systems, dynamic grid alignment, and contextual tool palettes that adapt based on the selected elements, ensuring precise placement while maintaining creative flexibility. Advanced features include multi-select operations, transform controls with visual handles, and instant preview updates that reflect changes across both 2D and 3D views simultaneously.

Automatic Code

The automatic code generation system translates visual designs into clean, optimized MythicMobs YAML format without requiring manual coding or syntax knowledge. The system intelligently groups particles by type, optimizes coordinate precision, and generates human-readable comments that document the effect structure for easy maintenance and debugging. Advanced features include automatic effect grouping, intelligent offset calculations, and configurable export settings that allow creators to fine-tune the output format for specific server configurations or performance requirements. The generated code maintains consistency across exports, enabling version control integration and collaborative development workflows where multiple creators can work on the same effect while preserving the visual intent and technical specifications.

3D Support

The integrated 3D editor provides comprehensive three-dimensional visualization and editing capabilities that complement the 2D canvas workflow. This advanced system allows creators to view effects from any angle, manipulate particles in true 3D space, and preview how effects will appear in actual game environments with proper depth and perspective. The 3D editor features professional-grade camera controls including orbit, pan, and zoom operations, transform gizmos for precise positioning, and real-time lighting that accurately represents particle visibility and color relationships. Seamless integration between 2D and 3D workflows enables creators to sketch concepts quickly in 2D for rapid iteration, then refine details in 3D for final positioning and camera choreography, providing the best of both worlds for efficient effect creation.

Quick start

  1. Create two layers: one for structure, one for accents. Name them clearly.
  2. Block out forms with Circle/Square/Line; add highlights with Free tool.
  3. Tune layer color, alpha and size for legibility; toggle visibility to compare.
  4. Add motion: a subtle global Rotate, plus a per-layer pulse or move.
  5. Iterate density until the silhouette reads at different zoom levels.
  6. Export MythicMobs YAML and review offsets to verify positions.
See also: Getting Started

Best practices

  • Use layers to isolate experiments and keep history clean.
  • Prefer batch edits (layer color/alpha) over per-element micromanagement.
  • Keep density modest; motion sells form more than raw point count.
  • Enable Performance Mode on heavy scenes; export quality is unaffected.
  • Save milestones after composition, styling and animation phases.
Learn more: Layers · Modes

Export pipeline

Export groups vertices by particle type, emits stable IDs and writes x/y/z offsets with controlled precision. Review the YAML once to validate origin alignment, then reuse the project as a living source of truth. Image/OBJ imports are best treated as references—simplify them into readable motion before exporting.
